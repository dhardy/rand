pub fn range<X: SampleRange, R: Rng + ?Sized>(low: X, high: X, rng: &mut R) -> X

Generate a random value in the range [low, high).

This is a convenience wrapper around Range. If this function will be called repeatedly with the same arguments, one should use Range, as that will amortize the computations that allow for perfect uniformity.

Panics

Panics if low >= high.

Example

use rand::distributions::range2::range;

let mut rng = rand::thread_rng();
let n: u32 = range(0, 10, &mut rng);
println!("{}", n);
let m: f64 = range(-40.0f64, 1.3e5f64, &mut rng);
println!("{}", m);
