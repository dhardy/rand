Generic trait for sampling random values from some distribution

TODO: quite possibly remove both this and SimpleRand since Sample is more convenient and distributions like Default handle all the real work.

Example

use rand::distributions::{Rand, Default, Range};
 
let mut rng = rand::thread_rng();
println!("Random byte: {}", u8::rand(&mut rng, Default));
println!("Random range: {}", i32::rand(&mut rng, Range::new(-99, 100)));

Required Methods

fn rand<R: Rng + ?Sized>(rng: &mut R, distr: D) -> Self

Generate a random value of the given type, according to the specified distribution.

The distributions Default (or Uniform and Uniform01) and Range should cover most simpler usages; Normal, LogNormal, Exp, Gamma and a few others are also available.
